Display Format Guide

Each format has three sections — left | middle | right — one per display.

  • dd — whole days
  • hh — hours within a day (0–23)
  • hhh — total hours; never wraps into days (a 30‑hour countdown shows 30)
  • mm / ss — minutes / seconds
  • u — tenths of a second
  • D H N — unit letters for days, hours, minutes; rendered lowercase on the display as d, h, n (n because a 7‑segment m isn't readable)

Overflow

  • A value needing 3+ digits drops its unit letter to make room: 45 d → 456.
  • hhh:mm in one section fits up to 99:59; above that it switches to the split hhh | mm layout automatically.
  • Leading sections that would show only zeros stay blank until needed.
